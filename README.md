# Task Tracker

Aplikasi web ini dapat digunakan untuk manajemen tugas

## Persyaratan

* PHP versi >8.0
* Composer
* MySQL

## Cara Instalasi

1. Clone repo projek
2. Ubah nama file "env" menjadi ".env"
3. Buka cmd dalam root folder projek
4. Jalankan ```composer install``` di dalam cmd tersebut

## Cara Konfigurasi Database

1. Buat database bernama "task_tracker"
2. Import file sql ke dalam database (lokasi file: database/task_tracker.sql)


## Cara Menjalankan Aplikasi

1. Buka cmd dalam root folder projek
2. Jalankan ```php spark serve``` di dalam cmd tersebut
